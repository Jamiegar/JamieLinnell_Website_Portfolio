Crash

Role:

Programming and Design

Game Description:

Crash is a fighting game based on Super Smash Bros. It implements many key features including hitstop, camera shake, unqiue abilities, haptic feedback and more. Using the Gameplay Ability System I was able to create many abilities and effects that are executed in a way so that the code is decoupled and maintainable. The project also contains a Character Creation Tool which is used to rapidly make new characters and abilties and then assign the abilities to the player input.

Mechanics and Systems

Character Abilities and Attrabutes

In this project all of the characters abilties, effects and attrbuties are created with the Gameplay Ability System (GAS). GAS is a framework in Unreal desinged to allow for complex abilties and effects to be created and link to eachother. It is a powerful framework that I used to develop the abilties. As the project takes inspiration from Super Smash Bros, I reseached into the abilties in Smash Bros and found that each ability could be narrowed down to 8 unquire types of abilities. The project has implmeted a base class of each abilitiy catorgory and has the tools to create more easily.

Character Creation Tool

The Character Creation Tool was made for this project to streamline the process of creating characters and mapping the abilities to inputs. The tool is made using Unreal's UI framework Slate in C++. The layout of the tool is shown in the image, it shows the key functionality of the tool. The tool can create charatcers and abilities in the content browser and those character can be customized however desired. The tool can then be used to select a character and choose which input should active what ability, this is done easily by clicking the input icon and selected the ability that input should activate.
